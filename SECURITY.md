## Reporting a Vulnerability

We take the security of our software seriously. If you suspect you have found a vulnerability, please report it immediately.

### How to Report

Please do **NOT** open a public issue on GitHub for security vulnerabilities.
Instead, please report the vulnerability by emailing us at:

junseopkim1228@gmail.com

### What to Include

- A description of the vulnerability.
- Steps to reproduce the issue.
- Any relevant code snippets or logs.

We will acknowledge your report within 48 hours and will work to resolve the issue as quickly as possible.
